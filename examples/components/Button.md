# Button Component

## Description
A reusable button component with customizable variants and states.

## Props
- variant: 'primary' | 'secondary' | 'outline'
- size: 'sm' | 'md' | 'lg'
- fullWidth: boolean
- disabled: boolean
- onClick: () => void
- children: ReactNode

## Examples
```tsx
<Button variant='primary' size='md'>
  Click me
</Button>
```
